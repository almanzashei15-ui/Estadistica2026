
About Dataset
Este dataset forma parte de un proyecto que busca determinar como influye emocionalmente emigrar a España, con un enfoque comparativo entre la ciudad andaluza de Córdoba y el municipio de Priego de Córdoba. Para este proyecto se realizó un análisis exhaustivo por medio de las siguientes herramientas: Microsoft Excel, SQL y Tableau. Se tomaron en cuenta las siguientes variables: el arraigo emocional, la profesión de los migrantes, su nivel educativo, nacionalidad, motivo de migración, la percepcion del arraigo y el tiempo de residencia.

El propósito de este proyecto es explorar el impacto emocional y de pertenencia que poseen los migrantes en Priego de Córdoba y Córdoba Capital, tomando en cuenta las variables ya descritas.

El dataset ha sido estructurado y visualizado mediante herramientas como SQL y Tableau. Las visualizaciones complementarias pueden consultarse en Tableau Public e incluye un dashboard que muestra lo siguiente:
